# Redis Check

## Instalasi

# Buat dan aktifkan virtual environment

```bash

python3 -m venv .venv
source .venv/bin/activate
```

# Copy dan isi konfigurasi

```bash

cp .env.example .env
# Edit .env: isi GITHUB_TOKEN, REDIS_HOST, REDIS_PORT, TARGET_ORG, dll
```

# Jalankan instalasi

```bash

bash install.sh
```
